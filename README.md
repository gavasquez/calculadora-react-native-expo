
# Calculator

## Getting Started

Proyecto para el curso de React Native de DevTalles, creado con Expo.

### Prerequisites

- Node.js
- Yarn

### Installation

1. Clone the repository

```bash
git clone https://github.com/corhuil/react-native-calculator.git
```

2. Install dependencies

```bash
npm install
```

3. Run the app

```bash
npm run start
```
